# TPO Lab 2 — Система функций с интегрированным тестированием

## 📋 Описание

Реализована система, состоящая из тригонометрической и логарифмической частей:

- **x ≤ 0:** `(((((sec(x) + sec(x)) / tan(x)) - csc(x)) * sin(x))²)` — ветка тригонометрии
- **x > 0:** Сложная логарифмическая функция с log₂, log₃, log₅, log₁₀

Все функции реализованы через разложение в ряды (Маклорена для sin, artanh для ln).

## 📈 Анализ функций

### Область определения

| Функция | Область | Ограничения |
|---------|---------|-------------|
| sin(x) | ℝ | — |
| ln(x) | (0, +∞) | x > 0 |
| log_a(x) | (0, +∞) | x > 0 |
| **TrigSystem** | (-∞, 0] | sin(x) ≠ 0, cos(x) ≠ 0, tan(x) ≠ 0 |
| **LogSystem** | (0, +∞) | x ≠ 1, знаменатели ≠ 0 |

### Недопустимые точки

- **TrigSystem:** x = 0, ±π, ±2π, ... (sin = 0); x = ±π/2, ±3π/2, ... (cos = 0)
- **LogSystem:** x = 1 (log = 0, приводит к делению на 0)

## 🔧 Детали реализации

### Sin(x) — Ряд Маклорена

```
sin(x) = Σ (-1)^n * x^(2n+1) / (2n+1)!  для n = 0, 1, 2, ...
```

С редукцией аргумента к [-π, π] для стабильности вычислений.

### Ln(x) — Ряд artanh

```
ln(x) = 2 * Σ ((x-1)/(x+1))^(2n+1) / (2n+1)  для n = 0, 1, 2, ...
```

Сходится для всех x > 0.

### Логарифмы с другими основаниями

```
log_a(x) = ln(x) / ln(a)
```

## 🧪 Примеры тестов

### SinTest

```kotlin
@Test
fun testSinPiHalf() {
    val result = sin.compute(PI / 2)
    assertEquals(1.0, result, tolerance, "sin(π/2) = 1")
}

@Test
fun testSinOddFunction() {
    val result1 = sin.compute(0.7)
    val result2 = sin.compute(-0.7)
    assertEquals(-result1, result2, tolerance, "sin(-x) = -sin(x)")
}
```

### LnTest

```kotlin
@Test
fun testLnPropertyLnAB() {
    // ln(a*b) = ln(a) + ln(b)
    val lnAB = ln.compute(a * b)
    val lnA = ln.compute(a)
    val lnB = ln.compute(b)
    assertEquals(lnAB, lnA + lnB, tolerance)
}
```

## 📝 Стратегия интеграции

### Bottom-Up подход

1. **Sin** (базовая) → тесты ✅
2. **Cos** (зависит от Sin) → тесты ✅
3. **Sec, Csc, Tan** (зависят от Sin, Cos) → тесты ✅
4. **TrigSystem** (использует все триг функции) → тесты ✅
5. **Ln** (базовая) → тесты ✅
6. **Log₂, Log₃, Log₅, Log₁₀** (зависят от Ln) → тесты ✅
7. **LogSystem** (использует все лог функции) → тесты ✅
8. **SystemFunction** (маршрутизация) → тесты ✅
