# TPO Lab 2 — Система функций с интегрированным тестированием

## Описание

Реализована система из тригонометрической и логарифмической частей:

- **x <= 0:** `(((sec(x) + sec(x)) / tan(x)) - csc(x)) * sin(x))^2`
- **x > 0:** логарифмическая функция с log2, log3, log5, log10

Все функции реализованы через разложение в ряды: ряд Маклорена для sin, ряд artanh для ln.

## Область определения

| Функция    | Область    | Ограничения                              |
|------------|------------|------------------------------------------|
| sin(x)     | R          |                                          |
| ln(x)      | (0, +inf)  | x > 0                                    |
| log_a(x)   | (0, +inf)  | x > 0                                    |
| TrigSystem | (-inf, 0]  | sin(x) != 0, cos(x) != 0                 |
| LogSystem  | (0, +inf)  | x != 1, знаменатели != 0                 |

Недопустимые точки:
- **TrigSystem:** x = 0, +-pi, +-2pi (sin = 0); x = +-pi/2, +-3pi/2 (cos = 0)
- **LogSystem:** x = 1

## Реализация

**sin(x)** — ряд Маклорена с редукцией аргумента к [-pi, pi]:
```
sin(x) = sum(-1^n * x^(2n+1) / (2n+1)!)
```

**ln(x)** — ряд через artanh, сходится для всех x > 0:
```
ln(x) = 2 * sum(((x-1)/(x+1))^(2n+1) / (2n+1))
```

**log_a(x)** — через натуральный логарифм:
```
log_a(x) = ln(x) / ln(a)
```
